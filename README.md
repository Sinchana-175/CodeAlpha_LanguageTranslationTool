# CodeAlpha Language Translation Tool

## Project Overview

This project is an AI-powered Language Translation Tool developed as part of the CodeAlpha Artificial Intelligence Internship.

The application allows users to translate text between multiple languages through a simple and interactive web interface.

## Features

* Translate text between multiple languages
* User-friendly Streamlit interface
* Supports English, Hindi, Kannada, Tamil, Telugu, Malayalam, French, German, and Spanish
* Fast and accurate translations using Google Translator

## Technologies Used

* Python
* Streamlit
* Deep Translator

## Installation

```bash
pip install -r requirements.txt
```

## Run the Application

```bash
streamlit run app.py
```
